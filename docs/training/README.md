## Certification Program

### Level 1: Basic User

**Requirements:**
- Complete onboarding
- Pass basic quiz (80%+)
- Complete 3 hands-on exercises

**Topics:**
- Account management
- Job creation
- Resume upload
- Basic matching

### Level 2: Advanced User

**Requirements:**
- Level 1 certification
- Pass advanced quiz (85%+)
- Complete 5 advanced exercises

**Topics:**
- Custom scoring
- Bias detection
- Bulk operations
- Advanced analytics

### Level 3: Administrator

**Requirements:**
- Level 2 certification
- Pass admin exam (90%+)
- Complete admin training

**Topics:**
- User management
- System configuration
- Monitoring and alerts
- Troubleshooting
