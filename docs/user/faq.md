## Resume Upload

### What file formats are supported?

- PDF (recommended)
- Microsoft Word (.doc, .docx)
- Plain text (.txt)

### What's the maximum file size?

10 MB per file. We recommend keeping files under 5 MB for faster processing.

### How long does processing take?

Typically 2-5 minutes per resume:
- **Parsing:** 30-60 seconds
- **Processing:** 1-2 minutes
- **Total:** 2-5 minutes

### Can I upload multiple resumes at once?

Yes! You can upload multiple files simultaneously. Each file is processed independently.

### What if a resume fails to process?

Check the error message, ensure the file is readable, and try reprocessing. Contact support if issues persist.

## Matching

### How does the matching work?

The system uses AI to:
1. Extract skills, experience, and education from resumes
2. Compare against job requirements
3. Calculate match scores
4. Rank candidates by fit

### What's the difference between matching strategies?

- **Standard:** Full analysis, best accuracy (2-5 min)
- **Fast:** Vector similarity, fastest (30-60 sec)
- **Comprehensive:** Standard + bias detection (5-10 min)

### How accurate is the matching?

Match scores are based on:
- Skills match (50% weight)
- Experience match (30% weight)
- Education match (20% weight)

Accuracy depends on resume quality and job requirements clarity.

### Can I customize the scoring weights?

Yes! Adjust weights in matching settings to emphasize what matters most for each role.

### What is bias detection?

Automated analysis that identifies potential biases in:
- Gender language
- Age-related terms
- Institution preferences

Provides recommendations for more inclusive hiring.
