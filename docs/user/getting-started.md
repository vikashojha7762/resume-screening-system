## Account Setup

### 1. Register an Account

1. Navigate to the registration page
2. Enter your email address
3. Create a strong password (minimum 8 characters)
4. Click "Sign Up"
5. Check your email for verification (if enabled)

### 2. Login

1. Go to the login page
2. Enter your email and password
3. Click "Sign In"
4. You'll be redirected to the dashboard

## First Steps

### Dashboard Overview

After logging in, you'll see the dashboard with:

- **Active Jobs** - Number of active job postings
- **Total Resumes** - Number of resumes in the system
- **Candidates Matched** - Number of matched candidates
- **Average Match Score** - Overall matching quality

### Navigation

Use the sidebar to navigate:

- **Dashboard** - Overview and statistics
- **Jobs** - Manage job postings
- **Resumes** - Upload and manage resumes
- **Candidates** - View matched candidates
- **Settings** - Account and system settings

## Creating Your First Job

### Step 1: Create Job Posting

1. Click **"Jobs"** in the sidebar
2. Click **"Create Job"** button
3. Fill in the job details:

   - **Title:** e.g., "Senior Software Engineer"
   - **Description:** Detailed job description
   - **Required Skills:** List of mandatory skills
   - **Preferred Skills:** Optional skills
   - **Minimum Experience:** Years of experience required
   - **Required Degree:** Education requirements

4. Click **"Save"**

### Step 2: Review Job Requirements

The system uses your job requirements to match candidates. Be specific:

- ✅ **Good:** "Python, FastAPI, PostgreSQL, 5+ years"
- ❌ **Vague:** "Programming experience"

## Uploading Resumes

### Supported Formats

- PDF (`.pdf`) - Recommended
- Microsoft Word (`.doc`, `.docx`)
- Plain Text (`.txt`)

### File Requirements

- Maximum size: **10 MB**
- Recommended: Under 5 MB for faster processing

### Upload Process

1. Go to **"Resumes"** page
2. Drag and drop files or click to select
3. Wait for upload confirmation
4. Processing starts automatically

### Processing Status

Resumes go through these stages:

1. **Uploaded** - File received
2. **Parsing** - Extracting text (30-60 seconds)
3. **Processing** - Analyzing content (1-2 minutes)
4. **Processed** - Ready for matching ✅

## Matching Candidates

### Start Matching

1. Go to **"Jobs"** page
2. Click on a job
3. Click **"Match Candidates"** button
4. Select matching strategy:
   - **Standard** - Full analysis (recommended)
   - **Fast** - Quick matching for large pools
   - **Comprehensive** - Detailed analysis with bias detection

5. Click **"Start Matching"**

### Matching Process

The system will:

1. Analyze all processed resumes
2. Calculate match scores based on:
   - Skills match (50% weight)
   - Experience match (30% weight)
   - Education match (20% weight)
3. Rank candidates by score
4. Detect potential biases
5. Generate detailed reports

**Processing time:** 2-5 minutes for 100 resumes

## Viewing Results

### Ranked Candidate List

After matching completes:

1. View the ranked candidate list
2. See overall scores (0-100%)
3. Review score breakdown:
   - Skills score
   - Experience score
   - Education score

### Candidate Details

Click on a candidate to see:

- **Score Breakdown** - Detailed scoring
- **Skills Radar Chart** - Visual skill comparison
- **Experience Timeline** - Work history
- **Education Details** - Academic background
- **Match Explanation** - Why they matched

### Filtering and Sorting

- Filter by minimum score
- Sort by rank, score, or date
- Search by candidate ID

## Tips for Best Results

### Job Postings

1. **Be Specific:** Clear requirements = better matches
2. **List Skills:** Include all required technologies
3. **Set Experience:** Specify minimum years needed
4. **Update Regularly:** Keep job descriptions current

### Resume Uploads

1. **Quality Matters:** Clear, well-formatted resumes parse better
2. **Complete Information:** Resumes with full details score higher
3. **Recent Resumes:** Upload fresh resumes for best results

### Matching

1. **Use Standard Strategy:** Best balance of speed and accuracy
2. **Review Bias Detection:** Check recommendations
3. **Adjust Weights:** Customize scoring if needed
4. **Export Results:** Download CSV/PDF for sharing
